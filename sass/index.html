    ratio_golden

    @function ratio_golden($size, $increment) { ... }

    Description

    Golden Ratio Function Calculates a size using the golden ratio (approximately 1.618), which is widely used in design for its aesthetically pleasing proportions.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $size

    The base size to scale.

    		Number— none
    $increment

    The exponent applied to the golden ratio, default is 1.

    		Number— none

    Returns

    Number —

    The calculated size based on the golden ratio.

    Example

    Usage Example

    // Increase size using the golden ratio
.element {
  width: golden_ratio(16px, 1); // Output: 25.888px
}

// Decrease size using the golden ratio
.small-element {
  width: golden_ratio(16px, -1); // Output: 9.888px
}

    Throws

    • The size parameter must be a valid number or unitless value.

    • The increment parameter must be a valid number.
